# Basic example

Minimal Mado API tour: `signal`, `computed`, `effect`, `component()`,
`html```, `routes()`, context, persisted state and lazy modules.

## Run

```bash
npm run build
npm run serve -- basic
```

Open <http://localhost:5173/>.

## Role

Use this example when you need the smallest readable app shape before moving to
the larger `tickets` or `showcase` examples.
