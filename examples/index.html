Mado — examples

Browser examples run one at a time and mount at the dev-server root, so the router behaves like it would in a production SPA.

Run

npm run build
npm run serve -- basic
npm run serve -- tickets
npm run serve -- showcase

Examples

  • basic

    Minimal tour: signals, components, router, context, persisted state and lazy modules.

    npm run serve -- basic

  • tickets

    LLM zero-history CRUD validation: resource, queryParam, computed, useForm, mutation, invalidates, each.

    npm run serve -- tickets

  • showcase

    Flagship SaaS CRM pressure app: marketing routes, auth admin, CRUD/workflows, context, forms and browser regression.

    npm run serve -- showcase

  • cloudflare

    Cloudflare Workers deployment/edge example. This is not a browser SPA for the dev server.

    cd examples/cloudflare

Without an argument, npm run serve shows this page. Through CLI: mado examples, mado dev showcase.

Add Example

  1. Create examples/<name>/index.html, main.ts and routes.ts.
  2. In index, load /dist/examples/<name>/main.js.
  3. Add a card here and verify npm run serve -- <name>.
